Retail Sales Performance & Profitability Analysis
📌 Project Overview

This project analyzes a retail sales dataset containing 9,994 records, including detailed information about orders, customers, products, regions, sales, profit, and discounts.

The purpose of this project is to demonstrate end-to-end Exploratory Data Analysis (EDA) using Python and extract meaningful business insights from raw transactional data.

🎯 Project Objectives

Understand overall sales performance

Identify the most and least profitable product categories

Compare regional sales and profit performance

Examine the impact of discount levels on profitability

Generate actionable business recommendations

🔎 Key Insights

Revenue is concentrated in a small number of high-value transactions

Technology is the highest revenue-generating category

Excessive discounting negatively impacts profitability

Regional performance varies significantly

High sales do not always translate into high profit

📈 Business Recommendations

Optimize discount strategy

Focus marketing on high-margin products

Re-evaluate pricing in loss-making categories

Strengthen performance in underperforming regions
